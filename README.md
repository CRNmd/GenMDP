# <p align="center">Exécution :</p>

Clonez le projet dans le répértoire de votre choix : 

```git clone https://github.com/CRNmd/GenMDP ```

Allez dans le répértoire du projet :

```cd GenMDP ```

Executez le fichier main.py : 

```python3 GenMDP.py```
